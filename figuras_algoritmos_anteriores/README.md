# Visualizações dos Testes — Explicação

Foram geradas várias imagens que registram experimentos realizados com diferentes estruturas de vizinhança e heurísticas construtivas. Cada conjunto de imagens é referente aos diferentes testes relizados:
- a vizinhança aplicada (ex.: troca simples, swap, permutacao, etc),
- a heurística construtiva inicial (ex.: gulosa ou construtiva aleatoria),
- diferentes formas de representacao via grafico

Observações sobre as imagens no diretório raiz da pasta de figuras:
- No root da pasta de figuras há uma versão das imagens com uma nova escala (zoom e escala de 10 em 10) para facilitar a leitura da convergência do algoritmo.
- Também deixei as anteriores lá com nomes descritivos nas pastas referentes a cada um delas.
- As imagens com a escala antiga (mais ampla inicial e com zoom de para escala de 0 À 10) estão nas pastas específicas de cada teste.
